# QCefBrowser

#### 介绍
基于CEF封装的Qt浏览器，支持音视频播放，开发环境vs2017+qt5.12.11

重构项目

1.基本的浏览器功能

2.支持音视频播放

3.支持与js交互

4.支持动画渲染

5.支持离屏渲染

6.支持svga
